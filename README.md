# SisyphusClicker

## Description

In this clicker, help Sisyphus push his boulder up Mount Olympus. He might have to start again, but he will succeed and beat Zeus !

Upgrade through a skill tree and climb the mountain even faster. By upgrades from Favors you gain by climbing or fighting in the coloseum !

Game made for Game Jam Connexion 2025
